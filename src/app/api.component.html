Following Api's Were Used in This Tracker

State Data

curl --location --request GET 'https://api.covidindiatracker.com/state_data.json' \ --data-raw '

District Data

curl --location --request GET 'https://api.covid19india.org/state_district_wise.json' \ --data-raw '

Tatal India Data

curl --location --request GET 'https://api.covidindiatracker.com/total.json' \ --data-raw '
